| Factor                               | Details                                                                                  | Recommended / Ideal Range                                   |
| ------------------------------------ | ---------------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| **Crawlability & Indexability**      | Ensure all important pages are crawlable, robots.txt is correct, no accidental `noindex` | All main pages indexable, no orphan pages                   |
| **XML Sitemap**                      | Helps search engines understand site structure                                           | Submit in Google Search Console; all important pages listed |
| **HTTPS / SSL**                      | Secure websites rank better                                                              | Must have HTTPS; certificates valid                         |
| **Canonicalization**                 | Avoid duplicate content issues                                                           | Each page has correct `<link rel="canonical">`              |
| **Robots.txt**                       | Correctly block unnecessary pages                                                        | Only block non-essential pages like admin, login            |
| **Structured Data / Schema**         | Helps Google understand content type                                                     | Implement JSON-LD; test via Rich Results Test               |
| **Mobile-friendliness**              | Google uses mobile-first indexing                                                        | Pass Google Mobile-Friendly Test; responsive layout         |
| **URL Structure**                    | Clean, readable URLs with keywords                                                       | Short, lowercase, hyphen-separated; < 75 characters         |
| **404 & Redirects**                  | Proper handling prevents ranking loss                                                    | 301 redirects for moved pages; minimal 404s                 |
| **Canonical & Duplicate Management** | Avoid content duplication                                                                | Use canonical tags, 301 redirects, avoid thin content       |
