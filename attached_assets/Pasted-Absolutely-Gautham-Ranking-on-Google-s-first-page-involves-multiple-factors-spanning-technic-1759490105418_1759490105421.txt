## **1. Technical SEO Factors**

These ensure that Google can **crawl, index, and understand your website** properly.

| Factor                               | Details                                                                                  | Recommended / Ideal Range                                   |
| ------------------------------------ | ---------------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| **Crawlability & Indexability**      | Ensure all important pages are crawlable, robots.txt is correct, no accidental `noindex` | All main pages indexable, no orphan pages                   |
| **XML Sitemap**                      | Helps search engines understand site structure                                           | Submit in Google Search Console; all important pages listed |
| **HTTPS / SSL**                      | Secure websites rank better                                                              | Must have HTTPS; certificates valid                         |
| **Canonicalization**                 | Avoid duplicate content issues                                                           | Each page has correct `<link rel="canonical">`              |
| **Robots.txt**                       | Correctly block unnecessary pages                                                        | Only block non-essential pages like admin, login            |
| **Structured Data / Schema**         | Helps Google understand content type                                                     | Implement JSON-LD; test via Rich Results Test               |
| **Mobile-friendliness**              | Google uses mobile-first indexing                                                        | Pass Google Mobile-Friendly Test; responsive layout         |
| **URL Structure**                    | Clean, readable URLs with keywords                                                       | Short, lowercase, hyphen-separated; < 75 characters         |
| **404 & Redirects**                  | Proper handling prevents ranking loss                                                    | 301 redirects for moved pages; minimal 404s                 |
| **Canonical & Duplicate Management** | Avoid content duplication                                                                | Use canonical tags, 301 redirects, avoid thin content       |

## **2. Website Speed & Performance**

Site speed affects ranking and user experience. Google uses **Core Web Vitals**.

| Factor                             | Metric                         | Recommended / Ideal Range                  |
| ---------------------------------- | ------------------------------ | ------------------------------------------ |
| **LCP (Largest Contentful Paint)** | Measures page load speed       | < 2.5s (good)                              |
| **FID (First Input Delay)**        | Measures interactivity         | < 100ms (good)                             |
| **CLS (Cumulative Layout Shift)**  | Measures visual stability      | < 0.1 (good)                               |
| **Time to First Byte (TTFB)**      | Server response time           | < 200ms optimal                            |
| **Page Size**                      | Total page load                | < 2MB recommended                          |
| **Requests**                       | Number of HTTP requests        | < 50–60 ideal                              |
| **Image Optimization**             | Proper size, format            | WebP/AVIF; compress without losing quality |
| **Lazy Loading**                   | Load images/videos when needed | Yes, especially for long pages             |
| **Caching & CDN**                  | Reduce server load and latency | Enable browser caching; use CDN globally   |

**Tools to check:** PageSpeed Insights, GTmetrix, Lighthouse.

## **3. On-Page SEO**

Optimizing content **for users and search engines**.

| Factor                       | Details                                   | Ideal/Benchmark                            |
| ---------------------------- | ----------------------------------------- | ------------------------------------------ |
| **Title Tag**                | Unique, relevant, includes target keyword | 50–60 characters                           |
| **Meta Description**         | Summarizes page, includes keyword         | 150–160 characters                         |
| **Header Tags (H1, H2, H3)** | Structure content                         | One H1 per page; H2/H3 for subsections     |
| **Keyword Usage**            | Include primary and secondary keywords    | Natural density: 1–2%                      |
| **Content Length**           | Longer content ranks better if valuable   | 1200–2500+ words (depends on topic)        |
| **LSI / Semantic Keywords**  | Related terms to improve relevance        | Include 5–10 related terms naturally       |
| **Internal Linking**         | Helps crawl & distributes link equity     | Link to relevant pages; 2–5 per page good  |
| **Outbound Links**           | Link to authority sites                   | 1–3 relevant authoritative links per page  |
| **Image Alt Text**           | Describe images for accessibility & SEO   | Include primary keyword if relevant        |
| **Content Freshness**        | Regular updates signal value              | Update important content every 6–12 months |
| **Multimedia**               | Videos, images, infographics              | Improves engagement and dwell time         |

## **4. Off-Page SEO**

How your site is **trusted and linked by other sites**.

| Factor                       | Details                              | Ideal Range / Goal                                     |
| ---------------------------- | ------------------------------------ | ------------------------------------------------------ |
| **Backlinks**                | Quality > quantity                   | 20–50 high-authority backlinks for small sites; DA 40+ |
| **Anchor Text**              | Relevant keywords in backlinks       | Natural mix: branded, exact match, partial             |
| **Social Signals**           | Shares, mentions                     | No direct ranking factor, but increases visibility     |
| **Brand Mentions**           | Unlinked mentions on authority sites | Helps Google associate authority                       |
| **Domain Authority / Trust** | Overall trust metric                 | DA 40+ recommended for competitive niches              |

---

## **5. User Experience (UX) & Engagement**

Google evaluates **behavioral signals**.

| Factor                | Metric                      | Ideal Range / Benchmark          |
| --------------------- | --------------------------- | -------------------------------- |
| **Bounce Rate**       | % of single-page sessions   | < 40% good                       |
| **Dwell Time**        | How long users stay         | > 2–3 mins for content pages     |
| **Pages per Session** | User engagement             | 2–4 pages/session minimum        |
| **Navigation & Menu** | Ease of finding content     | Clear menus, breadcrumbs         |
| **Mobile UX**         | Tap targets, readable fonts | Pass Google Mobile-Friendly Test |

## **6. Content Quality & E-A-T (Expertise, Authority, Trust)**

| Factor                         | Details                              | Recommended Action                           |
| ------------------------------ | ------------------------------------ | -------------------------------------------- |
| **Expert Content**             | Written by credible authors          | Include author bio, credentials              |
| **Authority**                  | Recognized in the niche              | Earn backlinks from authoritative sites      |
| **Trustworthiness**            | HTTPS, privacy policy, accurate info | Include contact page, terms, privacy         |
| **User Intent Match**          | Content matches query intent         | Informational / transactional / navigational |
| **Multimedia & Rich Snippets** | Tables, FAQ, reviews                 | Structured data to increase CTR              |

## **8. Advanced Technical Considerations**

* **Lazy Load for Images / Videos**: Speeds up LCP.
* **AMP (Optional)**: For news or content-heavy sites.
* **Server Location / CDN**: Faster delivery globally.
* **Hreflang Tags**: For multi-language sites.
* **Pagination / Infinite Scroll**: Use `rel="next/prev"` or proper SEO handling.
* **Core Structured Data Types:** Articles, FAQ, Product, Recipe, LocalBusiness.

**Technical + Content + Backlinks**, with UX and speed as amplifiers.
